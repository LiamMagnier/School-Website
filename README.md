# Primetel

A responsive, multi-page website built as a first-year Computer Science project. The brief required **Bootstrap**, so the layout is built on top of it, with a calm, modern look — a creamy background, a warm orange accent, subtle motion, and a light/dark theme switcher.

The design started in Figma (one of several concepts on my [Figma page](https://figma.com/@LiamMagnier)) and was then built out in HTML, CSS and JavaScript.

## Features

- 🌗 **Light / dark mode** switcher
- 📱 **Responsive** across mobile, tablet and desktop
- ✨ Hover effects, background blur and small animations
- 🎨 Consistent visual system — creamy background with an orange accent
- 🧩 Built with **Bootstrap** (project constraint) plus custom CSS

## Tech stack

- **HTML5** (74.8%)
- **CSS3** (23.7%)
- **JavaScript** (1.5%)
- **Bootstrap**

## Project structure

```
School-Website/
├── assets/imgs/    # images
├── css/            # custom styles (on top of Bootstrap)
├── javascripts/    # interactions (theme switch, etc.)
├── pages/          # additional pages
└── index.html      # entry point
```

## Run locally

A static site — no build step needed.

```bash
git clone https://github.com/LiamMagnier/School-Website.git
cd School-Website
# open index.html in your browser
```

> Tip: use the VS Code "Live Server" extension for live reload while editing.
